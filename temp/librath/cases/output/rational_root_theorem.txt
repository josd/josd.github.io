Rational Root Theorem:
If f(x)=a0 x^n + ... + an ∈ ℤ[x] and r=p/q in lowest terms is a root, then p|an and q|a0.
In particular, monic ⇒ every rational root is an integer.
========================================================================
REASON WHY
========================================================================
Let f(x) = a0 x^n + a1 x^{n-1} + ... + an with a0 ≠ 0, ai ∈ ℤ.
Suppose r = p/q in lowest terms (gcd(p,q)=1, q>0) and f(r)=0.
Multiply by q^n:
    0 = q^n f(p/q) = a0 p^n + a1 p^{n-1} q + ... + an q^n.
Rearrange:
    a0 p^n = - q( a1 p^{n-1} + a2 p^{n-2} q + ... + an q^{n-1} ).
Thus q | a0 p^n. Since gcd(p,q)=1, we get q | a0.
Similarly, rewrite as
    an q^n = - p( a0 p^{n-1} + a1 p^{n-2} q + ... + a_{n-1} q^{n-1} ),
so p | an. This proves:  p | an and q | a0.

Corollaries:
• If a0=1 (monic), then q | 1 ⇒ q=1, so every rational root is an integer.
• If an = ±1, then p | ±1 ⇒ p = ±1, likewise giving only integral possibilities.
