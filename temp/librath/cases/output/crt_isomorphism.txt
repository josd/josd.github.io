========================================================================
ANSWER
========================================================================
Chinese Remainder Theorem (ring form): for pairwise coprime m1,…,mk, the map
  φ: Z/MZ → ∏ Z/miZ,   φ([x]_M) = ([x]_{m1},…,[x]_{mk})
is a ring isomorphism with inverse x ≡ Σ r_i e_i (mod M), where e_i are CRT idempotents.
========================================================================
REASON WHY
========================================================================
Let m1,…,mk be pairwise coprime and M=m1⋯mk. For each i, set Mi=M/mi and choose
ti ≡ (Mi)^{-1} mod mi. Define ei = Mi·ti (an integer, considered mod M). Then:
  • ei ≡ 1 (mod mi) since Mi·ti ≡ 1 (mod mi), and  ei ≡ 0 (mod mj) for j≠i because mj|Mi.
  • Σ ei ≡ 1 (mod M).
For any residue data (r1,…,rk) with ri ∈ Z/miZ, the element
    x ≡ Σ ri·ei  (mod M)
satisfies x ≡ ri (mod mi) for all i, so it reconstructs the class in Z/MZ.
This gives an explicit inverse to φ, hence φ is bijective. Additivity and multiplicativity
hold because both φ and reconstruction are Z-linear in the residue coordinates and
the ei’s are fixed modulo M; thus φ is a ring isomorphism.
